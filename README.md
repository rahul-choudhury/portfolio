# workspace

A Bun + Turborepo monorepo for personal apps, shared UI, and tooling.

## Apps

- `apps/portfolio`: [rchoudhury.dev](https://rchoudhury.dev) personal portfolio built with Next.js
- `apps/bookmarks`: [bookmarks.rchoudhury.dev](https://bookmarks.rchoudhury.dev) personal link manager built with Next.js + PostgreSQL

## Packages

- `packages/design-system`: shared UI primitives and design tokens
- `packages/biome-config`: shared Biome config
- `packages/typescript-config`: shared TypeScript config
